##### 题目描述

给定一个文本文件 file.txt，请只打印这个文件中的第十行。

示例:

假设 file.txt 有如下内容：

Line 1
Line 2
Line 3
Line 4
Line 5
Line 6
Line 7
Line 8
Line 9
Line 10
你的脚本应当显示第十行：

Line 10
##### 思路

先使用最简单的逐行扫描读取，count等于10时输出

##### 关键点解析

无

##### 代码

```bash
# Read from the file file.txt and output the tenth line to stdout.
#！bin/bash

count=0 
while read line;do
((count++))
if [ $count -eq 10 ];then 
echo $line 
fi
done < file.txt
```
